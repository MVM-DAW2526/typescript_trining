# Gestió d'Inventari 📦

## Calculadora de Valor Total del Carret

**Objectiu:** Crear estructures de dades complexes sense ajuda.

**Requeriments:**

- Crea una `interface` anomenada `Producte`. Ha de contenir: `id` (número), `nom` (text), `preu` (número) i `quantitat` (número).
    
- Crea un Array que contingui com a mínim 3 productes diferents amb diferents quantitats i preus.
    
- Crea una funció anomenada `calcularTotal` que rebi aquest array de productes com a paràmetre.
    
- La funció ha de calcular el preu total (sumant el preu multiplicat per la quantitat de cada producte) i **retornar un número**.
    
- Imprimeix el resultat final per consola amb un missatge clar, per exemple: "El total del carret és: 45.50€".
